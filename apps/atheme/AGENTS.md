# Atheme (IRC Services)

> Scope: IRC services app. Inherits monorepo [AGENTS.md](../../AGENTS.md).

Atheme IRC services (NickServ, ChanServ, OperServ, MemoServ). No justfile mod; runs as Docker service via `infra/compose/irc.yaml`.

## Tech Stack

Atheme · Docker · config templates (env substitution via `scripts/prepare-config.sh`)

## Repository Structure

```
config/
├── atheme.conf.template   # Config template (env vars substituted at init)
└── atheme.conf            # Generated (gitignored)

Containerfile
docker-entrypoint.sh
```

## Config Generation

Config is generated by `scripts/prepare-config.sh` (run via `just init`). Uses `${VAR}` from `.env`. After editing `.env`, rerun `./scripts/prepare-config.sh`.
